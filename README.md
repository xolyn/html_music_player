# html_music_player
An HTML music player with realtime lyrics display 

## change song and lyrics
Search for (`CTRL+F`) `hh.mp3` in `index.html` and `hh.lrc` in `js.js` and modify.

## demo
see a demo [here](https://f.xoy.one/test/)
